# Signals

Synthesized, cross-case-study patterns, and the `a11y-signals.yml` schema those patterns feed into.

- [`review-patterns-v0.1.md`](review-patterns-v0.1.md) — patterns across the case studies.
- [`a11y-signals.schema.yml`](a11y-signals.schema.yml) — schema v0.1 (draft). A machine-readable file any project can place at its repo root, analogous to CODEOWNERS, to declare its accessibility posture: conformance target, review capacity, and AT test matrix.
- [`examples/`](examples/) — two worked examples: this repository's own (honest, including its gaps) and an illustrative higher-maturity example inferred from case-study evidence.

Not yet built: a validator tool for this schema (planned for a later phase — see [`../roadmap.md`](../roadmap.md)).
